☁️ iCloud Inbox Workflow

No browser needed on your phone. Just drop files into iCloud Drive and VidNugget processes them automatically.

Inbox is being watched
Checking every 5 seconds for new files.
Step 1 — Drop a URL into your iCloud inbox

On your iPhone, open the Files app and navigate to:
iCloud Drive → VidNugget → inbox

Create a new text file (or use the Shortcut below) containing the YouTube URL, e.g.:
https://youtube.com/watch?v=dQw4w9WgXcQ

VidNugget will pick it up within 5 seconds.
Step 2 — Optionally add screenshots

While watching the video on your phone, take screenshots of key moments.
Save them into the same folder:
iCloud Drive → VidNugget → inbox

Drop them in before or alongside the URL file. Claude Vision will read all of them and fold that context into the nugget.
Step 3 — Find your nugget

When done (usually 30–60 seconds), the nugget appears in:
iCloud Drive → VidNugget → knowledge_base → {Category} → {video title}

Open nugget.md in any markdown reader on your iPhone (Notion, Bear, Obsidian, or just the Files preview). The URL .txt is moved to inbox/done/ automatically.
iPhone Shortcut (one-tap from share sheet)

1. Open the Shortcuts app → New Shortcut
2. Add action: Receive → URLs
3. Add action: Text → set content to Shortcut Input
4. Add action: Save File → iCloud Drive → VidNugget → inbox
  • File name: video-[Current Date].txt
  • Overwrite: On
5. Name it "Send to VidNugget" and add it to the share sheet.

Now in YouTube → Share → "Send to VidNugget" → done. No typing needed.
