DROP FILES HERE
===============

VidNugget watches this folder every 5 seconds.

To process a video without opening the browser:
  1. Create a .txt file containing a YouTube URL, e.g.:
       https://www.youtube.com/watch?v=dQw4w9WgXcQ

  2. Save/drop it into this folder.

  3. VidNugget will process it automatically and save the
     knowledge nugget to knowledge_base/

This works great with:
  - iCloud Drive (share from iPhone → save here)
  - AirDrop to this folder
  - iPhone Shortcut that saves URLs to this folder
